Installation
-------------

**Linux**
    
    ::

        git clone https://github.com/HexaOneOfficial/Elcix-OS.git 
        
 move the dotfiles to the home directory       

       

    
   
**Elcix os**
    
    ::

        gc https://github.com/HexaOneOfficial/Elcix-OS.git 
        
 move the dotfiles to the home directory
