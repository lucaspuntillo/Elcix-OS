Installation
-------------

**Linux**
    
    ::

        git clone https://github.com/HexaOneOfficial/Elcix-OS.git 
        
 move the dotfiles from the distro folder to the home directory       

       

    
   
**Elcix os**
    
    ::

        gc-elcix
        
 move the dotfiles from the distro folder to the home directory
